Python Scipy模块功率谱密度相关

利用python实现对振动信号的功率谱分析

记录一下在使用python实现信号的功率谱密度相关计算时，所参考的一些文档的链接：

Scipy官方文档

自功率谱密度函数——periodogram

互功率谱密度——csd

相干分析函数——coherence

Note:

上述函数的输入都是一个ndarray的数组，不是矩阵。例如：

输入的数组可以为：x1 = [0 1 1 3 4 5 6 6]
输入的数组为：x2 = [[0] [1] [1] [3] [4] [5] [6] [6]]，则会导致计算结果全为0。
这是因为x1和x2的维度不一样。

可以验证一下：

x1 = [0, 1, 1, 3, 4, 5, 6, 6]
x2 = [[0], [1], [1], [3], [4], [5], [6], [6]]
print('x1维度: ', np.ndim(x1))
print('x2维度: ', np.ndim(x2))

输出结果为：

x1维度: 1
x2维度: 2

因此，一定要十分注意数据输入的格式是否符合要求！
